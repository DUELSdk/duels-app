# CycleDuel — UI & Animation Spec

---

## Animation Spec

**Feel:** Heavy and grinding. Each block is a read war. Three chances to break even.

| Moment | Animation |
|--------|-----------|
| Peek | Opponent's first card flips face-up center screen, brief hold — feels like a challenge thrown down |
| Lock | Hands slide in from both sides, settle into position |
| Clash (×3 per block) | Heavier than Card Duel — slower slide into center, more spring resistance, bigger shockwave ring, winner travels further, loser recoil more definitive |
| Block end | Used cards fade out, remaining hands pull back slightly. Short breath before next block. |
| Block 3 bench | Benched card slides off to the side and grays out before lock |
| Match end | Winner holds forward, loser retracted, fade out |

Heavier than Card Duel because there are only 3 clashes per block — each one counts more.
