# Card Duel — UI & Animation Spec

---

## Animation Spec

**Feel:** Cold and surgical. You laid your trap before the match started. Now watch it play out.

| Moment | Animation |
|--------|-----------|
| Lock | Cards slide in from both sides, settle into position |
| Reveal | Both cards flip face-up simultaneously |
| Clash | Both slide toward center — small flash + subtle shockwave ring at impact, winner pushes through, loser recoils. Spring physics. |
| Score tick | Quiet, no fanfare |
| Next slot | Brief pause, repeat |
| Match end | Winner side holds forward, loser fully retracted, fade to result |

Effects: present but restrained — enhance the clash, don't perform it. No impact frames.
